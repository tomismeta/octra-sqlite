# Release Manifests

This directory contains machine-readable release manifests for audited,
network-specific artifacts.

`octra-sqlite-0.2.0.json` is the current release manifest checked by
`octra-sqlite status`.

`octra-sqlite-0.1.0.json` records the original blessed Circle WASM artifact used
by the default `octra-sqlite new` path. The `0.2.0` Rust CLI/library release
keeps the same Circle WASM bytes and hash.
