# Release Manifests

This directory contains machine-readable release manifests for audited,
network-specific artifacts.

`octra-sqlite-0.2.1.json` is the current release manifest checked by
`octra-sqlite status`.

`octra-sqlite-0.1.0.json` records the original blessed Circle WASM artifact used
by the default `octra-sqlite new` path. The `0.2.0` and `0.2.1` Rust
CLI/library releases keep the same Circle WASM bytes and hash.
